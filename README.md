## auto-files-bkup

### Build process

```bash
    $ cd build && cmake .. && make
    $ sudo make install
```

### Installation process
Clone this repository first. Then do this.
```bash
    $ sudo ./install.sh
```

### How to use
After installation, you can do this to use the application.
```bash
    $ auto_files_bkup /path/to/backup 3 # 3 being the delay between one backup to next.
```
